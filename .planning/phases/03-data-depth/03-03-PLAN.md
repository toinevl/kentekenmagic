## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-03-01 | Tampering | Plate param in recall/modification queries | mitigate | Plate normalized to 6 alphanumeric chars upstream; passed as query param value not interpolated |
| T-03-03-02 | Tampering | t49b-isb7 recall status response | mitigate | recallStatusRowSchema.parse() validates shape; .passthrough() preserves extras without executing |
| T-03-03-03 | Tampering | j9yg-7rg9 recall details response | mitigate | recallDetailRowSchema.parse() validates; display fields (omschrijving_defect, beschrijving_van_het_herstel) are text-only strings rendered in React (no innerHTML / dangerouslySetInnerHTML) |
| T-03-03-04 | Tampering | sghb-dzxx modification response | mitigate | rdwModificationSchema.parse() validates; demontagedatum "0" handled by explicit string equality |
| T-03-03-05 | Tampering | RDW "moreInfoUrl" field injecting external links | mitigate | meer_informatie_op_internet value passed through as string; frontend must render as text or use rel="noopener noreferrer" on any anchor |
| T-03-03-06 | Denial of Service | Recall detail parallel fetches (one per recall status row) | mitigate | Entire fetch() wrapped in 3.5-second Promise.race by aggregator; source cannot block indefinitely |
