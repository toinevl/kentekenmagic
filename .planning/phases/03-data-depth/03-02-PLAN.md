| API → RDW Socrata (sgfe-77wx, a34c-vvps, hx2c-gt7k) | External HTTPS calls; responses are untrusted |
| RDW response → application state | Zod schema validates all three datasets before data touches application logic |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | Tampering | Plate parameter passed to fetchRdwDataset | mitigate | Plate is normalized to 6 alphanumeric chars before reaching this source; passed as query param value, not interpolated into URL path or SQL |
| T-03-02-02 | Tampering | RDW sgfe-77wx response content | mitigate | rdwInspectionRowSchema.parse() with Zod validates shape and types; .passthrough() preserves unknown fields without executing them |
| T-03-02-03 | Tampering | RDW a34c-vvps + hx2c-gt7k defect responses | mitigate | rdwDefectRowSchema and rdwDefectDescSchema validate each row; gebrek_omschrijving is display text only, not eval'd |

| T-03-02-05 | Information Disclosure | RDW API errors logged | mitigate | fetchRdwDataset throws on non-2xx; aggregator catches and logs error message only (no plate or response body per established pattern) |
